How it works

Three steps. Then never again.

Set up once. Autofill any form, on any website, forever.

1

Create a profile

Add your name, email, phone, address, and any custom fields. Create one profile for job applications, another for school — or link a profile to a specific website.

2

Browse to any form

TETRAform detects forms automatically. The assistant bar appears at the top of the page — always visible when you need it, invisible when you don't.

3

One click — done

Click "Fill from profile" and watch every field fill in seconds. Review and submit. ⚡ 23 fields in 4 seconds.

Power user? Use Ctrl + Shift + F to autofill without opening the popup. (⌘+Shift+F on Mac)

FAQ

Frequently asked questions

Is TETRAform free?
Yes — free to install and use. The free plan includes up to 3 profiles, 15 form fills per day (resets at midnight), and all core features. A Pro plan with unlimited fills and profiles is coming soon. No credit card, no sign-up required.
Is my data safe?
Completely. All profile data is stored in your browser's local storage — it never touches a server. Not ours, not anyone's. No cloud sync, no account, no data collection. You can export or delete everything at any time.
Does it work on all websites?
TETRAform works on the vast majority of standard web forms — job boards, scholarship portals, CRMs, contact forms, and more. You can also choose to ignore specific websites where you don't want the assistant to appear.
What's the difference between profiles?
Each profile stores a different set of information. Create a "Job Applications" profile with your resume details, a "School" profile with academic info, and a "Personal" profile with your home address. Switch in one click — or link a profile to a website so it activates automatically.
Do I need to create an account?
No. TETRAform works without an account — no email, no password, no sign-up. Install it, create a profile, and start filling forms. Your data lives in your browser.
Can I use it on a new computer?
Yes. Use the Backup & Restore feature to export your profiles to a file. Install TETRAform on your new machine and import the file — all your profiles are back instantly.
